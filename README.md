# zmq
ZMQ cpp/java demo

# Dependencies

# 1 - ZMQ Library

 > sudo apt-get install libtool autoconf automake libsodium-dev

 > git clone https://github.com/zeromq/libzmq.git

 > cd libzmq

 > ./autogen.sh

 > ./configure

 > make

 > sudo make install

 > sudo mv /usr/local/lib/*zmq* /lib

 * Installed files:

 > file /lib/*zmq*
